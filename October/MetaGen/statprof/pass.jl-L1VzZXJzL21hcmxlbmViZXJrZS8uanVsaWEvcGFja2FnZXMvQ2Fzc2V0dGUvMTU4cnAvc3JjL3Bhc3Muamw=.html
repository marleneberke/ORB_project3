5 """
6 ```
7 Cassette.@pass transform
8 ```
9
10 Return a Cassette pass that can be provided to the `Context` constructor's `pass` keyword
11 argument in order to apply `transform` to the lowered IR representations of all methods
12 invoked during contextual execution.
13
14 `transform` must be a Julia object that is callable with the following signature:
15
16 ```
17 transform(::Type{<:Context}, ::Cassette.Reflection)::Union{Expr,CodeInfo}
18 ```
19
20 If `isa(transform(...), Expr)`, then the returned `Expr` will be emitted immediately
21 without any additional processing. Otherwise, if `isa(transform(...), CodeInfo)`, then
22 the returned `CodeInfo` will undergo the rest of Cassette's overdubbing transformation
23 before being emitted from the `overdub` generator.
24
25 Two special `Expr` heads are available to Cassette pass authors that are not normally valid
26 in Julia IR. `Expr`s with these heads can be used to interact with the downstream built-in
27 Cassette passes that consume them.
28
29 - `:nooverdub`: Wrap an `Expr` with this head value around the first argument in an
30 `Expr(:call)` to tell downstream built-in Cassette passes not to overdub that call. For
31 example, `Expr(:call, Expr(:nooverdub, GlobalRef(MyModule, :myfunc)), args...)`.
32
33 - `:contextslot`: Cassette will replace any `Expr(:contextslot)` with the actual `SlotNumber`
34 corresponding to the context object associated with the execution trace. For example, one
35 could construct an IR element that accesses the context's `metadata` field by emitting:
36 `Expr(:call, Expr(:nooverdub, GlobalRef(Core, :getfield)), Expr(:contextslot), QuoteNode(:metadata))`
37
38 Cassette provides a few IR-munging utility functions of interest to pass authors; for details,
39 see [`insert_statements!`](@ref), [`replace_match!`](@ref), and [`is_ir_element`](@ref).
40
41 Note that the `@pass` macro expands to an `eval` call and thus should only be called at
42 top-level. Furthermore, to avoid world-age issues, `transform` should not be overloaded after
43 it has been registered with `@pass`.
44
45 Note also that `transform` should be "relatively pure." More specifically, Julia's compiler
46 has license to apply `transform` multiple times, even if only compiling a single method
47 invocation once. Thus, it is required that `transform` always return a generally "equivalent"
48 `CodeInfo` for a given context, method body, and signature. If your `transform`
49 implementation is not naturally "pure" in this sense, then it is still possible to guarantee
50 this property by memoizing your implementation (i.e. maintaining a cache of previously
51 computed IR results, instead of recomputing results every time).
52
53 See also: [`Context`](@ref), [`overdub`](@ref)
54 """

75 Return `x` with all subelements `y` replaced with `replace(y)` if `ismatch(y)`. If
76 `!ismatch(y)`, but `y` is of type `Expr`, `Array`, or `SubArray`, then replace `y`
77 in `x` with `replace_match!(replace, ismatch, y)`.
78
79 Generally, `x` should be of type `Expr`, `Array`, or `SubArray`.
80
81 Note that this function modifies `x` (and potentially its subelements) in-place.
82
83 See also: [`insert_statements!`](@ref), [`is_ir_element`](@ref)
84 """
85 function replace_match!(replace, ismatch, x)
86 if ismatch(x)
87 return replace(x)
88 elseif isa(x, Array) || isa(x, SubArray)
89 for i in eachindex(x)
90 x[i] = replace_match!(replace, ismatch, x[i])
91 end
92 elseif isa(x, Expr)
93 replace_match!(replace, ismatch, x.args)
94 end
95 return x
96 end

107
108 For every insertion, all downstream `SSAValue`s, label indices, etc. are incremented
109 appropriately according to number of inserted statements.
110
111 Proper usage of this function dictates that following properties hold true:
112
113 - `code` is expected to be a valid value for the `code` field of a `CodeInfo` object.
114 - `codelocs` is expected to be a valid value for the `codelocs` field of a `CodeInfo` object.
115 - `newstmts(stmt, i)` should return a `Vector` of valid IR statements.
116 - `stmtcount` and `newstmts` must obey `stmtcount(stmt, i) == length(newstmts(stmt, i))` if
117 `isa(stmtcount(stmt, i), Int)`.
118

120 say our `code` object contains several statements:

125 ```
