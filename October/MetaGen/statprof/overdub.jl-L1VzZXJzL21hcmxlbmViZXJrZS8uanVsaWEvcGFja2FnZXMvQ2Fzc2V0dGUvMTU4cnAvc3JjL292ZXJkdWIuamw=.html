616 @doc(
617 """
618 ```
619 overdub(context::Context, f, args...)
620 ```
621
622 Execute `f(args...)` overdubbed with respect to `context`.
623
624 More specifically, execute `f(args...)`, but with every internal method invocation `g(x...)`
625 replaced by statements similar to the following:
626
627 ```
628 begin
629 prehook(context, g, x...)
630 overdub(context, g, x...) # %n
631 posthook(context, %n, g, x...)
632 %n
633 end
634 ```
635
636 Otherwise, if Cassette cannot retrieve lowered IR for the method body of `f(args...)`,
637 then `fallback(context, f, args...)` will be called instead. Cassette's [`canrecurse`](@ref)
638 function is a useful utility for checking if this will occur.
639
640 If the injected `prehook`/`posthook` statements are not needed for your use
641 case, you can disable their injection via the [`disablehooks`](@ref) function.
642
643 Additionally, for every method body encountered in the execution trace, apply the compiler pass
644 associated with `context` if one exists. Note that this user-provided pass is performed on
645 the method IR before method invocations are transformed into the form specified above. See
646 the [`@pass`](@ref) macro for further details.
647
648 If `Cassette.hastagging(typeof(context))`, then a number of additional passes are run in
649 order to accomodate tagged value propagation:
650
651 - `Expr(:new)` is replaced with a call to `Cassette.tagged_new`
652 - `Expr(:splatnew)` is replaced with a call to `Cassette.tagged_splatnew`
653 - conditional values passed to `Expr(:gotoifnot)` are untagged
654 - arguments to `Expr(:foreigncall)` are untagged
655 - load/stores to external module bindings are intercepted by the tagging system
656
657 The default definition of `overdub` is to recursively enter the given function
658 and continue overdubbing, but one can interrupt/redirect this recursion by
659 overloading `overdub` w.r.t. a given context and/or method signature to define
660 new contextual execution primitives. For example:
661
662 ```
663 julia> using Cassette
664
665 julia> Cassette.@context Ctx;
666
667 julia> Cassette.overdub(::Ctx, ::typeof(sin), x) = cos(x)
668
669 julia> Cassette.overdub(Ctx(), x -> sin(x) + cos(x), 1) == 2 * cos(1)
670 true
671 ```

673 See also: [`recurse`](@ref), [`prehook`](@ref), [`posthook`](@ref)
